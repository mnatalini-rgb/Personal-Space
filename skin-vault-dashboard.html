5. Experiment Decision Framework

Key Questions

Is organic traffic sufficient?

Current: 22,734 views in 41 days (~554/day avg). Second traffic spike from Apr 14 — now averaging ~1,500/day in last week.

Do users engage beyond the tab click?

9.9% click inventory items (1,422 of 14,326). Down from 12.8% at Apr 14 — second spike brought more casual browsers.

Is sentiment positive?

94% non-negative (n=18). Only 1 'notMyThing' in 18 submissions.

Weekend traffic spike?

Second major spike from Apr 14. Traffic doubled from ~700/day to ~1,500/day. Cause TBD (possible expanded rollout or external driver).

Next Steps

  1. Resolved: Mar 24 — BETA badge (~3x uplift sustained)
  2. NEW: Investigate Apr 14 traffic spike (~2x uplift, cause TBD)
  3. NEW: Two new interaction types discovered: view_on_steam_market_click (77 users) and inspect_in_game_click (24 users) — add to product metrics framework
  4. Click rate dropped from 12.8% → 9.9% as traffic scaled — investigate if new traffic segment has different engagement pattern
  5. Feedback growing (18 submissions). 3 new responses since Apr 14. Still overwhelmingly positive.
  6. Monitor Steam API rate limiting (Isabel's dashboard)
  7. Decide: expand to more profiles or iterate on UX

Go/No-Go Criteria

GREEN

>2,000 views/week organic
>20% item click rate
>50% positive feedback

YELLOW

1,000-2,000 views/week
10-20% click rate

RED

<1,000 views/week
<10% click rate
Negative feedback dominant

6. Backlog & Ideas

Feature requests and improvement ideas collected from user feedback and team review. Prioritise during next sprint planning.

UX Improvements

Rarity Filter

Add a new filter by item rarity so users can quickly narrow inventory to the tier they care about (e.g., Mil-Spec, Restricted, Classified, Covert, Contraband).

Filter

Skin Name Search

Add a text search/filter by skin name so users can find specific items instantly instead of scrolling through the full inventory.

Filter

Functionality

Skin Image Links → Public URL

Clicking a skin image should redirect the user to the public Steam market listing or inspect link, rather than being a dead-end.

Link

Pin / Group Skins

Let users pin or select a custom group of skins (e.g., "my trade list", "favourites") for quick access and comparison — similar to wishlisting.
