5. Experiment Decision Framework

Key Questions

Is organic traffic sufficient?

Current: 16,976 views in 33 days (~515/day avg, sustained post-BETA badge)

Do users engage beyond the tab click?

12.8% click inventory items (1,407 of 10,989).

Is sentiment positive?

93% non-negative (n=15). First "notMyThing" received (1 of 15).

Weekend traffic spike?

Weekend traffic strong — sustained ~3x uplift from BETA badge (Mar 24). Traffic now averaging ~515 views/day across the full experiment period.

Next Steps

  1. Resolved: Mar 24 traffic spike — caused by "BETA" badge added to the Skin Vault tab, increasing discoverability (~3x uplift sustained through Apr 14)

  3. Feedback sample still growing (15 submissions). Analyse qualitative themes — first "notMyThing" response received.
  4. Monitor Steam API rate limiting (Isabel's dashboard)

6. Backlog & Ideas

Feature requests and improvement ideas collected from user feedback and team review. Prioritise during next sprint planning.

UX Improvements

Rarity Filter

Add a new filter by item rarity so users can quickly narrow inventory to the tier they care about (e.g., Mil-Spec, Restricted, Classified, Covert, Contraband).

Filter

Skin Name Search

Add a text search/filter by skin name so users can find specific items instantly instead of scrolling through the full inventory.

Filter

Functionality

Skin Image Links → Public URL

Clicking a skin image should redirect the user to the public Steam market listing or inspect link, rather than being a dead-end.

Link

Pin / Group Skins

Let users pin or select a custom group of skins (e.g., "my trade list", "favourites") for quick access and comparison — similar to wishlisting.
