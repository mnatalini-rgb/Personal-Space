Methodology

Experiment Setup & Key Decisions

Setup

  • Platform: CSStats.gg — 1.2M daily users, 9.3M MAU
  • A/B split: 50/50 via GA experiment (ads_ab_variant)
  • Duration: 20 days (25 Feb – 16 Mar 2026)
  • Page views: Moba 3.59M / Publift 3.52M (excl. Russia)

Data Sources

  • GA4 page views by experiment cohort & geo
  • GAM: Moba ad units (2 IDs)
  • GAM: Publift ad unit (1 ID)
  • Internal tier classification (T1: 8, T2: 33, T3: 90)

Key Decisions

  • Russia excluded — GAM does not operate there; 2.6M views removed from both cohorts
  • Fill rate rejected — Publift uses a GAM placeholder line item for unfilled impressions, inflating fill rate to ~100%
  • RPV as primary metric — Revenue per Page View is the only apples-to-apples comparison
  • Publift commission — ~15% deducted from gross to show true publisher revenue
  • US presented separately — 234x anomaly flagged for investigation
