1. Experiment Design

Problem: In long-term partnerships, account linkage rates decline over time, limiting partner value and reducing users entering the funnel.

Hypothesis: A "mystery reward" framework uses curiosity and variable-reward psychology to increase motivation. Users perceive a free, low-effort opportunity with potential upside.

Key Insights

Cost Flag: Equal reward distribution across all tiers (including highest-value Premium subs at 24%) inflated variant costs. Backend now supports weighted probabilities to control CPA.

Compliance Note: Account linking does not involve payment from users. No cost to participate — does not constitute gambling.

Next Steps

  1. Calibrate reward weights to reduce CPA (weight lower tiers more heavily)
  2. Evaluate Mystery Box as the default mechanic for all linkage missions
  3. Test Mystery Box vs Drops to compare gamification approaches
  4. Follow-up experiment running with Tradeit — per-challenge level (deeper funnel)

Scaling Note

Winline's eligible pool is ~450K active players (RU). At full scale, the annualised incremental value could reach ~€194K for Winline alone. Tradeit's eligible pool spans 169K+ users across multiple regions — with Mystery Box already driving their March 2026 spike to €68.7K (5× prior months).

Downstream Caveat

These projections are based on AL value only (top of funnel). The full partner value depends on downstream conversion quality (KYC, FTD, trades). Mystery Box drives volume, but Q2 priorities include tightening the conversion ladder to ensure AL uplift translates to proportional C1/C2 value.
