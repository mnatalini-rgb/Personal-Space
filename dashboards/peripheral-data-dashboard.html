Section D: Design Questions to Resolve

1. Separate or Combined?

Should we show peripherals separately or as a combined 'Setup' card? Since users often have a keyboard and mouse from different brands, a single "Gear" section makes more sense than isolated fields.

2. Handling Missing Data

What do we do when brand is unknown/undetected? Options include hiding the field completely, showing "Not detected", or just displaying the device type without a brand name.

3. Display CPU Info?

Data: 100% fill rate on CPU

Is it useful for the profile? Pro: shows hardware power. Con: clutters the profile. CPU strings need heavy formatting before display.

4. Do we need brand logos?

It is highly feasible to create logo assets for top Tier 1 brands, and fallback to simple text for the others to save design time.
