The Faceit Platform's rewards distribution system currently allows users to redeem vouchers, coupons, or prizes within the mission screen they are participating in. However, this setup has led to malicious users exploiting the system by playing games solely to redeem prizes and then selling them illicitly on unauthorised platforms.

This behavior undermines the effectiveness of advertisers' efforts to capture leads, as the prizes intended for their target audience end up in the hands of individuals who are not genuinely interested in their products or services. Consequently, advertisers invest resources in creating prizes that do not serve their intended purpose.

To address this issue, we need came up with the account linkage product which won’t allow users to handle codes. 

**Benefits**

The account linkage is platform based and will cover any competition & drops inside Faceit with one single sign-in

User will be able to claim their rewards only if they link their Faceit account with their advertiser. This has varius benefits for the user and advertiser:

1. **Auto-Claim**: Users must only link the accounts and any prize will be auto-claimed on the advertiser platform  
2. **Automated Process**: Streamlined user experience with automation, minimising friction by requiring only one-time account linkage.  
3. **Incentivise Account Creation**: Encourages users to create an account on the advertiser's platform.  
4. **Guaranteed Account Creation**: Leveraging the Fear of Missing Out by making reward acquisition contingent on account creation.  
5. **Prevent Code Reselling**: Users are shielded from seeing the code number, reducing the likelihood of code reselling.  
6. **Fraud Prevention**: Establishes a direct 1:1 relationship between the user and partner, mitigating fraudulent activities.

The account linkage feature requires an integration between Faceit and the advertisers which might take **up to 2 weeks** including testing.  
Technical documentation: [https://docs.faceit.com/getting-started/authentication/account-linkage/\#authorization](https://docs.faceit.com/getting-started/authentication/account-linkage/#authorization)

**How It Works**

Users can participate to the competition but it won’t be able to claim any prizes unless it has linked their Faceit account with the advertiser platform.

The operation of linking the account can be done from the competition main page or the order page.

We can track from which competition the users has linked their accounts

**Type of  prizes**

1. Advertiser vouchers ie.  
   1. free bets  
   2. raffle tickets  
   3. credits  
2. Embedded codes in a URL (example: [winline.ru/stories/?raffle=asfqwsfq?guid=ewfwef23](http://winline.ru/stories/?raffle=asfqwsfq?guid=ewfwef23))  
3. Material Prizes (the advertiser is responsible for the delivery)  
4. Faceit Points  
5. Faceit Premium Subscriptions  
* Faceit products can be also delivered behind mandatory advertiser account linkage.  
* Despite we can distribute raffle tickets, Faceit **cannot** run any raffles competition due to legal requirements, the raffle needs to be done on the advertisers side
