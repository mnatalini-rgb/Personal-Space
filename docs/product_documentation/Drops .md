**Drops in gaming**

In video gaming, a drop prize is a reward that is given to players for completing certain tasks or objectives. These rewards can be anything from in-game items to real-world prizes, such as gift cards or even cash. Drop prizes are a great way to incentives players to keep playing and to encourage them to achieve certain goals. They can also be used to promote new content or features in a game.

**Faceit Drops**

Drops serve to enhance engagement with Branded Events, thereby amplifying Brand Impressions (each drop equates to 3 brand impressions).

This feature functions as a mechanism for dispensing complimentary prizes to users who actively participate in Branded Events. Drops are exclusively allocated to users involved in these events.

Prizes can be supplied by the advertiser, but they can also purchase Faceit Points to augment the number of drops.

Drops are available for branded events only

**How It Works**

Even though the Drops distribution is random, the algorithm will try to distribute as the drops to as players possible and reward players that are more active.

**User Journey**

1\. **Distribution**: will happen at the post match page, in this way we can be sure that the user was active as it just completed a game session. We use a Faceit component called “rider” and we also send an activity notification

 

2\. **Claim**: at this stage the drop is only allocated which means that the user must manually claim it (click on the claim button). 

* The user will have 12h to claim the drop and it can be done from the rider or from the [Your Drop page](https://www.faceit.com/en/settings/dropshttps://www.faceit.com/en/settings/drops) (a countdown is also displayed)  
* After the drop expires, the drop goes back into the mix prizes and redistributed via another drop

3\. **Claim Confirmation**: once the user has claim it, the claim confirmation module will appear to confirm the action

**Key Features**

**Top Tiers Drops**

These special drops are distributed only at specific times of the day, standing out as rare occurrences in the users' perception. They are strategically employed to entice users to engage at least once daily throughout the entire duration of the competition.

The Faceit admin will label them as top tier during the setup and a count down will inform the user when the drops will be available for distribution.

**Account Linkage**

This competition support the account linkage feature. Please checkout the Account Linkage one pager for more informations

**Type of prizes**

1. Advertiser vouchers ie.  
   1. free bets  
   2. raffle tickets  
   3. credits  
2. Embedded codes in a URL (example: [winline.ru/stories/?raffle=asfqwsfq?guid=ewfwef23](http://winline.ru/stories/?raffle=asfqwsfq?guid=ewfwef23))  
3. Material Prizes (the advertiser is responsible for the delivery)  
4. Faceit Points  
5. Faceit Premium Subscriptions

**Glossary**

| Assigned (ro Pending) | the drops is been allocated to a users and has 6h before it expires |
| :---- | :---- |
| Claimed (or Paid) | user has claimed (clicked on the button) the drop and it belongs to the user |
| Expired (or Not Claimed) | the user didn’t claim the assigned drop on time and the drop is now ready be re-assigned to another user |
| Total Pool | the total drops available for this specific Branded Event |
| Drop Quantity | it’s the batches of FP that will be distributed with a single drop ie. 10, 50, 100, etc |
