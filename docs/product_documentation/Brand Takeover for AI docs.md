Ideal for brands that are after Awareness, Product launches and Brand Recognition.

Ladder-style competitions where gamers compete against other players to improve their ranking and win prizes on dedicated day(s) of the week.

The main KPI that drive success for this product are the **Brand Impressions**, each event will collect an average of 9-20\* brand impressions per unique player. \**this depends on varius factors including: geo, duration, drops and match room skin*

A **Brand Impression** occurs each time a user views the advertiser's logo. This impression is always guaranteed as it won't appear alongside other content (e.g., inside the matchmaking page). These impressions are not counted as Brand Impressions.

More Branded Event examples [here](https://drive.google.com/drive/folders/1BXBee-tp8Ezwxe_Tg642FrOK_6RdzvuZ)

**How It Works**

These ladders typically run for 7 days (weekly) and are geo-exclusive, meaning we cannot simultaneously run two branded events in the same geographic location. This restriction ensures fairness, as the same user could potentially collect points for both events using the same match.

The Branded Event calendar reservation can be found [here](https://docs.google.com/spreadsheets/d/1G9VHBM1H600-g_z68UdeYXbYMFTc4Hw0pPRQkk_fn8w/edit?usp=sharing).  
 Please contact **Irene Tamayo** for any questions

**User Experience**

1. **Upcoming event**: the branded event stands out from the normal ladders because we promote the event before it starts. Users will see the feature card on the Home and the Matchmaking page. This is to create buzz ahead of the event.  
2. **Auto-Join**: Users must be active on the platform (play at least one match) after the event has started, and it will automatically join the ladder. Users have no option to manually leave the branded event  
3. **Post Match Modal**: After joining the users will be informed to have joined the competition with a modal on the post match page  
4. **Post Match Modal** **Update**: after the first modal, users will then see another modal and it will only update them about the raking  
5. **Organiser Page**: Branded Events (future, current and past) can be also visualised from the organiser page

**Key Features**

**Prizes Expiration Time**

Advertisers have the option to set an expiration time for when users can claim their prizes. This feature ensures that users can still claim prizes even after the mission has ended. By setting an expiration time, advertisers can prevent users from claiming prizes after the campaign expiration, maintaining campaign integrity and user satisfaction.

**Match Skin**

A skin is a branded image that replaces the background of a website. It literally means putting a temporary skin for an advertiser over the top of your website’s design (hence the term re-skin). Often sold as part of a Sponsorship or Homepage Takeover, these ads can be very effective if costly. Reskins are always essentially co-branding exercises.

The Match Skin is composed by:

* Colour change of the element (same hex colour used by the BE)  
* Background image  
* Match sponsored by

When a BE is activated the users will automatically see the Match Skin on their queue in the different spots

Premium users are excluded from this feature until further review

2\. Match Prepared  \+ Voting 

3\. Match Finished

**Prizes**

Despite the Branded Event is mostly to drive brand recognition, prizes are mandatory and still a fundamental incentive for players.

Prizes are allocated by ranking and we recommend to reward between 10-20% of the forecasted number total of players, this is to make sure users are incentives to engage with the event. The top 3 prizes are highlights, also the 4-10th position

The prizes are distribution 24th after the competition end and users will be able to claim the rewards from the order page

**Type of  prizes**

1. Advertiser vouchers  
   1. free bets  
   2. raffle tickets  
   3. credits  
2. Material Prizes (the advertiser is responsible for the delivery)  
3. Faceit Points  
4. Faceit Premium Subscriptions  
5. Faceit Skins  
* Faceit products can be also delivered behind mandatory advertiser account linkage.  
* Despite we can distribute raffle tickets, Faceit **cannot** run any raffles competition due to legal requirements, the raffle needs to be done on the advertisers side
